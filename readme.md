## 提醒事項


### 啟動
1. 安裝套件
    使用 `pip`
    ``` bash
    pip install -r requirements.txt
    ```
   或是使用 `poetry`
    ``` bash
    poetry install
    ```
2. 啟動
    ``` bash
    python main.py
    ```
